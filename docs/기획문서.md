## 3. 예치금 제도

- **입금 계좌**: 하나은행 776-910015-28704 (산방에프앤비)
- **흐름**:
  1. 가맹점이 본사 계좌로 입금
  2. 관리자에게 문자 알림 수신
  3. 관리자가 **수동으로** 해당 가맹점의 예치금 잔액 조정
  4. 예치금 범위 내에서 가맹점이 발주 가능
- **자동 입금 연동**: 현재 단계에서는 미적용 (추후 고려)

### ⚠️ 예외: 대한상공회의소점 (직영점)
- 산방에프앤비와 **같은 법인** → 예치금 제도 미적용
- **후불 정산 방식**: 예치금 한도 없이 자유롭게 발주
- **[공급]** 신화푸드 → 상공회의소점(사업자B)에 직접 계산서 발행
- **[대금]** 상공회의소점 → 신화푸드에 직접 입금 (월말 외상)
- **[수수료]** 신화푸드 ↔ 산방에프앤비는 별도 정산
- 시스템에서 직영점 플래그로 구분하여 예치금 차감 없이 발주 처리

## 4. 사용자 및 권한

### 4-1. 관리자 (본사)
- 가맹점 계정 생성 (회원가입은 관리자만 가능)
- 예치금 잔액 수동 조정
- 발주 현황 관리
- 상품 관리
- **재고 관리** (입고 / 출고 / 재고 현황)
- **사용량 대비 잔여 재고 확인** → 발주 시점 놓치지 않도록
- 취소/파손/조정 **최종 승인**

### 4-2. 가맹점
- 로그인 후 발주
- 예치금 잔액 확인
- 발주 내역 확인 (본인 것만)
- 취소 요청 / 수량 변경 요청 / 클레임 접수
- ⚠️ **직접 회원가입 불가** — 관리자가 등록

### 4-3. 신화푸드 (배송 + 범용상품 관리)
- 발주 리스트 확인 (전체 가맹점)
- 배송 완료 처리
- 파손/불량 보고
- ✅ **범용상품 등록/수정** (상품명, 가격, 단위, 규격 등)
- ❌ 전용상품 관리 권한 없음
- ❌ 예치금 관련 정보 열람 불가
- ❌ 취소/수량 조정 권한 없음

### 4-4. 권한 매트릭스

| 기능 | 가맹점 | 신화푸드 | 관리자 |
|------|:------:|:-------:|:------:|
| 발주 하기 | ✅ | - | ✅ |
| 발주 리스트 보기 | 본인만 | ✅ 전체 | ✅ 전체 |
| 취소 요청 | ✅ 요청 | - | ✅ 승인 |
| 수량 변경 요청 | ✅ 요청 | - | ✅ 승인 |
| 파손/클레임 접수 | ✅ 접수 | ✅ 보고 | ✅ 처리 |
| 배송 완료 처리 | - | ✅ | ✅ |
| 예치금 조정 | - | - | ✅ |
| 전용상품 관리 | - | - | ✅ |
| **범용상품 등록/관리** | - | **✅** | ✅ |
| 재고 관리 | - | - | ✅ |

> **핵심 원칙**: 예치금에 영향을 주는 모든 행위 → 관리자만 최종 처리
> **범용상품**은 신화푸드가 소싱하므로, 상품 등록/가격/단위 설정은 신화가 직접 관리

## 7. 주문 / 출고 기준

### 7-1. 서울 · 내륙 지역

| 항목 | 기준 |
|------|------|
| 주문 마감 | **전일 오후 17시** |
| 출고일 | **월, 수, 금** (주 3회) |
| 배송 | 출고 당일 |

**예시:**
| 출고일 | 주문 마감 시점 |
|--------|--------------|
| 월요일 출고 | **일요일** 오후 17시까지 주문 |
| 수요일 출고 | **화요일** 오후 17시까지 주문 |
| 금요일 출고 | **목요일** 오후 17시까지 주문 |

> 마감 이후 주문 → 다음 출고일에 반영

### 7-2. 제주도 지역

| 항목 | 기준 |
|------|------|
| 주문 마감 | **출고일 D-2 오후 17시** |
| 배송 횟수 | **주 1회** |
| 배송 요일 | (확인 필요) |

**예시 (배송일이 수요일인 경우):**
| 출고일 | 주문 마감 시점 |
|--------|--------------|
| 수요일 출고 | **월요일** 오후 17시까지 주문 |

### 7-3. 시스템 구현 사항
- 마감 시간(17시) 이후 주문 → 자동으로 다음 출고일로 배정
- 가맹점 발주 화면에 **"다음 출고일"** 및 **"주문 마감까지 남은 시간"** 표시
- 지역별(서울·내륙 / 제주) 출고 스케줄 관리자가 설정 가능

## 8. 핵심 기능

### 8-1. 가맹점 발주
- 상품 목록 조회 및 수량 선택 후 발주
- 예치금 잔액 확인 / 발주 내역 확인

### 8-2. 재고 관리 (관리자)
- **입고**: 제조사로부터 산방푸드시스템즈로 입고 기록
- **출고**: 가맹점 발주에 따른 출고 기록
- **재고 현황**: 현재 품목별 재고 수량
- **사용량 예측**: 최근 사용량 대비 현재 재고로 얼마나 더 사용 가능한지 표시 → 제조사 발주 시점 알림

### 8-3. 월말 마감 / 세금계산서 관리 (관리자)

- **마감 기준: 출고일** (주문일 아님!)
  - 예: 3월 31일 주문 → 4월 1일 출고 → **4월 마감분**
- 가맹점별 월별 출고 내역을 한눈에 확인 가능해야 함
- **엑셀 다운로드 기능** 필수

#### 전용상품 vs 범용상품 구분

| 구분 | 전용상품 | 범용상품 |
|------|---------|---------|
| **뭐?** | 왕만두, 생밀면 등 7종 | 계란, 야채 등 |
| **누가 파나?** | **산방에프앤비 → 가맹점** | **산방에프앤비 → 가맹점** |
| **계산서 발행** | **산방에프앤비 → 가맹점** | **산방에프앤비 → 가맹점** |
| **소싱** | 산방푸드시스템즈 (제조사 매입) | 신화푸드 (자체 소싱) |
| **배송** | 신화푸드 (대행) | 신화푸드 |

> **모든 상품을 산방에프앤비가 가맹점에 판매** → 계산서도 산방에프앤비가 발행

#### 월말 계산서 체크리스트 (→ 상세 내용은 `거래관계구조.md` 참조)

| # | 방향 | 내용 |
|---|------|------|
| ① | 산방에프앤비 → 가맹점 | 전용+범용 판매 계산서 **(발행)** |
| ③ | 산방푸드시스템즈 → 산방에프앤비 | 전용상품 공급 계산서 (받음) |
| ④ | 신화푸드 → 산방에프앤비 | 배송수수료 계산서 - 전용 판매가의 10% (받음) |
| ⑤ | 신화푸드 → 산방에프앤비 | 범용상품 공급 계산서 - 범용 판매가의 97% (받음) |

> ⚠️ **직영점(상공회의소점) 예외**
> - 같은 법인이므로 **① 산방에프앤비 → 가맹점 계산서 발행 불필요** (내부 거래)
> - **신화푸드가 상공회의소점(사업자B)에 직접 계산서 발행** → 상공회의소점이 신화에 직접 입금 (월말 외상)
> - 신화푸드 ↔ 산방에프앤비 간 수수료는 별도 정산
> - 엑셀에서 직영점/가맹점 구분하여 추출 가능해야 함

#### 엑셀 추출 항목
| 항목 | 설명 |
|------|------|
| 주문일 | 가맹점이 발주한 날짜 |
| 출고일 | 실제 출고된 날짜 (마감 기준) |
| 가맹점명 | 발주한 가맹점 |
| 직영/가맹 구분 | 직영점(후불) / 가맹점(예치금) |
| 상품 구분 | 전용 / 범용 |
| 상품명 | 발주 상품 |
| 수량 | 발주 수량 |
| 단가 | 가맹점 판매가 |
| 금액 | 수량 × 단가 |
| 공급가액 | 부가세 제외 금액 |
| 부가세 | VAT |
| 합계 | 공급가액 + 부가세 |

> 이 데이터로 세금계산서 발행 및 월말 정산 처리
> 직영점 필터링 → 계산서 발행 대상에서 제외, 신화 후불 정산 금액 산출
