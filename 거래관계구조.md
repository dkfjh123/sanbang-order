## 2. 핵심 원칙

> **모든 상품(전용+범용)은 산방에프앤비가 가맹점에 판매한다.**
> **가맹점의 돈은 산방에프앤비로 들어온다. (예치금)**
> **신화푸드에는 돈을 직접 보내지 않는다. (통제 리스크)**

### ⚠️ 예외: 대한상공회의소점 (직영점)
- 산방에프앤비와 **같은 법인** (사업자번호만 다름)
- **예치금 미적용** → 한도 없이 자유롭게 발주
- **후불 정산**: 월마감 후 직영점 발주 금액을 산방에프앤비가 신화푸드에 입금

## 3. 전용상품 거래 구조

### 흐름
```
[상품]  제조사 → 산방푸드시스템즈 → (신화푸드 보관) → 가맹점 배송
[돈]    가맹점 → 산방에프앤비 (예치금)  ※직영점은 예치금 없음
[정산]  산방에프앤비 → 산방푸드시스템즈 (공급대금)
        산방에프앤비 → 신화푸드 (배송수수료)
```

### 마진 예시 (제품 1개 기준)
```
제조사 매입가:              10,000원
산방푸드 → 산방에프앤비 공급가: 12,000원  (산방푸드 마진 2,000원)
산방에프앤비 → 가맹점 판매가:  15,000원  (산방에프앤비 총마진 3,000원)

여기서:
  신화푸드 배송수수료:  15,000 × 10% = 1,500원
  산방에프앤비 순마진:  3,000 - 1,500 = 1,500원
```

| 주체 | 마진 |
|------|------|
| 산방푸드시스템즈 | 2,000원 (공급 마진) |
| 산방에프앤비 | 1,500원 (판매 마진 - 배송수수료) |
| 신화푸드 | 1,500원 (배송수수료) |

### 계산서
| 발행자 | 수신자 | 내용 |
|--------|--------|------|
| 제조사 | 산방푸드시스템즈 | 원자재 매입 |
| 산방푸드시스템즈 | 산방에프앤비 | 전용상품 공급 (12,000원) |
| **산방에프앤비** | **가맹점** | **전용상품 판매 (15,000원)** |
| 신화푸드 | 산방에프앤비 | 배송대행 수수료 (판매가의 10%) |

## 4. 범용상품 거래 구조

### 흐름
```
[상품]  신화푸드가 소싱 → 가맹점 배송
[돈]    가맹점 → 산방에프앤비 (예치금)  ※직영점은 예치금 없음
[정산]  산방에프앤비 → 신화푸드 (범용 판매가의 97% 정산)
```

### 마진 예시 (범용상품 월 매출 100만원 기준)
```
가맹점 예치금에서 차감:     1,000,000원 (이미 산방에프앤비에 있음)
신화푸드에 정산 (97%):     -970,000원
산방에프앤비 수익 (3%):     30,000원
```

| 주체 | 마진 |
|------|------|
| 산방에프앤비 | 판매가의 3% (로열티) |
| 신화푸드 | 판매가의 97% - 본인 매입원가 = 실질 마진 |

### 계산서
| 발행자 | 수신자 | 내용 |
|--------|--------|------|
| 신화푸드 | 산방에프앤비 | 범용상품 공급 (판매가의 97%) |
| **산방에프앤비** | **가맹점** | **범용상품 판매 (판매가 100%)** |

## 5. 🧾 월말 마감 정산 체크리스트

> 기준: **출고일** (주문일 아님! 3/31 주문 → 4/1 출고 = 4월분)

### 산방에프앤비가 해야 할 일

#### 📤 계산서 발행 (매출)
| # | 대상 | 내용 | 금액 기준 |
|---|------|------|----------|
| ① | 가맹점 (각각) | 전용상품 판매 | 해당 월 전용상품 출고 합계 |
| ② | 가맹점 (각각) | 범용상품 판매 | 해당 월 범용상품 출고 합계 |

> 💡 ①②는 하나로 합쳐서 발행해도 됨 (가맹점별 월 합계)
> ⚠️ 직영점(상공회의소점)은 같은 법인이므로 **계산서 발행 불필요** (내부 거래)

#### 📥 계산서 받기 (매입/비용)
| # | 보내는 곳 | 내용 | 금액 기준 |
|---|----------|------|----------|
| ③ | 산방푸드시스템즈 | 전용상품 공급 | 공급가 기준 합계 |
| ④ | 신화푸드 | 배송대행 수수료 | 전용상품 판매가의 10% |
| ⑤ | 신화푸드 | 범용상품 공급 | 범용상품 판매가의 97% |

#### 💰 돈 보내기 (정산)
| # | 보내는 곳 | 금액 | 비고 |
|---|----------|------|------|
| A | 산방푸드시스템즈 | 전용상품 공급대금 | 계산서 ③ 기준 |
