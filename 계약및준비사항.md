## 2. 계약 ①: 전용상품 공급계약

> **산방푸드시스템즈 → 산방에프앤비**

### 계약 핵심
- 산방푸드시스템즈가 제조사로부터 매입한 전용상품을 산방에프앤비에 공급
- 산방에프앤비가 가맹점에 판매

### 포함 내용
- [ ] 공급 대상 품목 (전용상품 7종)
- [ ] 품목별 공급가격 (매입가 + 산방푸드 마진)
- [ ] 공급가격 변경 절차 (원가 변동 시)
- [ ] 대금 정산 방식 및 주기 (월 정산 등)
- [ ] 품질 기준 및 클레임 처리
- [ ] 계약 기간 및 갱신 조건

### 세금계산서
```
산방푸드시스템즈 ──(세금계산서)──→ 산방에프앤비
  (공급가 기준, 예: 12,000원/건)
```

### ⚠️ 주의사항
- 관계사 거래이므로 **공급가가 시가 수준**으로 적정해야 함
- 부당행위 계산 부인 리스크 방지 (세법상 관계사 간 거래 규정)
- 공급가 산정 근거 문서화 필요

## 3. 계약 ②: 배송대행 계약

> **산방에프앤비 ↔ 신화푸드** (전용상품)

### 계약 핵심
- 산방에프앤비가 가맹점에 판매하는 전용상품의 보관 및 배송을 신화푸드에 위탁
- 신화푸드는 산방에프앤비의 배송대행자 역할

### 포함 내용
- [ ] 대행 업무 범위 (보관, 피킹, 배송)
- [ ] 수수료: **가맹점 판매가(부가세 포함)의 10%**
- [ ] 수수료 정산 주기 (월 정산)
- [ ] 배송 지역 및 배송 스케줄
- [ ] 상품 보관 조건 (냉동/냉장)
- [ ] 파손/분실 시 책임 범위
- [ ] 발주 알림 수신 및 확인 의무
- [ ] 계약 해지 시 상품 반환 절차
- [ ] 계약 기간 및 갱신 조건

### 세금계산서
```
신화푸드 ──(세금계산서)──→ 산방에프앤비
  (배송대행 수수료, 예: 판매가 15,000원의 10% = 1,500원/건)
```

---

## 4. 계약 ③: 범용상품 공급 계약

> **산방에프앤비 ↔ 신화푸드** (범용상품: 계란, 야채 등)

### 계약 핵심
- 신화푸드가 범용상품을 자체 소싱하여 **산방에프앤비에 공급** (판매가의 97%)
- **산방에프앤비가 가맹점에 판매** (판매가의 100%)
- 산방에프앤비는 3% 마진 확보

### 포함 내용
- [ ] 범용상품 범위 정의 (전용상품 외 모든 식자재)
- [ ] 공급가: **가맹점 판매가의 97%**
- [ ] 정산 주기 (월 정산) — 예치금에서 97%를 신화에 정산
- [ ] 범용상품 가격 결정 권한 (신화푸드 제안 → 산방에프앤비 승인?)
- [ ] 품질 기준 및 클레임 시 책임
- [ ] 계약 기간 및 갱신 조건

### 세금계산서
```
신화푸드 ──(세금계산서)──→ 산방에프앤비
  (범용상품 공급, 가맹점 판매가의 97%)

산방에프앤비 ──(세금계산서)──→ 가맹점
  (범용상품 판매, 판매가 100%)
```

## 5. 전체 세금계산서 흐름 요약

### 전용상품 (매월)
```
제조사 ──(계산서)──→ 산방푸드시스템즈          매입
산방푸드시스템즈 ──(계산서)──→ 산방에프앤비     전용상품 공급
산방에프앤비 ──(계산서)──→ 가맹점              전용상품 판매
신화푸드 ──(계산서)──→ 산방에프앤비            배송대행 수수료 (판매가의 10%)
```

### 범용상품 (매월)
```
신화푸드 ──(계산서)──→ 산방에프앤비            범용상품 공급 (판매가의 97%)
산방에프앤비 ──(계산서)──→ 가맹점              범용상품 판매 (판매가 100%)
```

### 정리: 산방에프앤비 월말 정산

#### 받는 것 (매출)
| 대상 | 내용 | 기준 |
|------|------|------|
| 가맹점 | 전용+범용 판매 대금 | 예치금에서 이미 차감됨 (별도 수금 불필요) |

#### 줘야 하는 것 (매입/비용)
| 대상 | 내용 | 기준 |
|------|------|------|
| 산방푸드시스템즈 | 전용상품 공급 대금 | 공급가 (예: 12,000원/건) |
| 신화푸드 | 배송대행 수수료 | 전용상품 판매가의 10% |
| 신화푸드 | 범용상품 공급 대금 | 범용상품 판매가의 97% |

> 💡 신화푸드에 줄 돈 = 배송수수료 + 범용공급대금 → 합산하여 한 번에 정산 가능

## 6. 기타 준비사항

### 산방에프앤비 ↔ 산방푸드시스템즈
- [ ] 관계사 거래 시가 적정성 검토 서류
- [ ] 사업자등록증 상호 확인
- [ ] 대금 정산용 계좌 지정

### 산방에프앤비 ↔ 신화푸드
- [ ] 배송대행 계약서 (전용상품)
- [ ] 범용상품 공급 계약서
- [ ] 정산 관련 실무 합의 (마감일, 정산일, 정산서 양식)
- [ ] 발주프로그램 접속 계정 및 권한 협의

### 산방푸드시스템즈 ↔ 제조사
- [ ] 기존 매입 계약 확인 (한만두식품, 다선, 다담푸드, 123푸드)
- [ ] 계약서 갱신 필요 여부 확인

### 세무 관련
- [ ] 세무사에게 전체 거래 구조 검토 요청
- [ ] 관계사 거래(산방푸드→산방에프앤비) 적정 공급가 자문
- [ ] 부가세 신고 시 각 법인별 매입/매출 정리 방법 확인
