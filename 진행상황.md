## 완성된 기능 (✅)

### 1. 로그인/인증
- 산방식당 로고 포함 로그인 페이지
- 역할별 자동 리다이렉트 (관리자/가맹점/신화푸드)
- 관리자만 계정 생성 가능 (가맹점은 직접 회원가입 불가)
- **이메일 기억하기** 체크박스 (localStorage 저장)
- Supabase 세션 유지 (브라우저 닫아도 자동 로그인)

### 2. 대시보드
- 역할별 다른 화면 (관리자: 통계카드, 가맹점: 예치금/발주, 신화: 처리대기)
- **제주 파레트 현황** 표시 (전용상품 55박스 = 1파레트, 프로그레스 바)
  - 관리자, 신화푸드, 제주 가맹점 모두에게 표시
  - 차단 아님, 인지용 (발주는 자유롭게 가능)
- 최신 공지사항 5건 미리보기
- 반응형 사이드바 (모바일 햄버거 메뉴)
- 관리자: 입금 확인 대기 목록 (승인/반려)
- 가맹점: 배송 스케줄 안내 (마감 카운트다운)

### 3. 가맹점관리 (관리자 전용)
- 가맹점 8곳 목록 (직영/가맹 구분, 권역, 예치금 잔액)
- 계정 생성 모달 (역할 선택, 소속 가맹점 연결)

### 4. 상품관리 (관리자 + 신화푸드)
- 전용상품 7종 + 범용상품 26종 등록 완료
- 전용상품: 매입가(제조사→산방푸드시스템즈) + 판매가(→가맹점) 모두 입력됨
- 범용상품: 신화푸드가 추가/수정/삭제 가능 (발주 이력 있는 상품은 삭제 불가 → 판매중지 이용)
- 판매중지 기능 (체크 해제 시 발주화면에서 안 보임, 상품관리에서 흐리게 표시)
- 전용/범용 필터 탭
- **가격 자동계산**: 부가세포함가 입력 시 공급가/부가세 자동 계산 (면세상품은 전액 공급가)
- **비밀번호 재인증**: 범용상품 가격 변경 시 신화푸드 비밀번호 재확인 필요

### 5. 발주하기
- 가맹점 선택 (관리자) 또는 자동 설정 (가맹점 로그인 시)
- 가맹점 미선택 시 상품 목록 비활성화
- 상품 검색 + 전용/범용 필터
- +/- 버튼으로 수량 선택 (큰 글씨)
- 하단 고정 바 (수량 뱃지 + 합계 + 발주하기 버튼)
- 상세 패널 (아래서 위로 펼침, 개별 수량 조절/삭제/전체 삭제)
- 예치금 잔액 확인 → 부족 시 발주 차단
- 직영점(대한상공회의소점): 예치금 없이 발주 가능
- **최소발주금액 15만원** (프론트 + API 양쪽 검증)
- **제주 파레트 현황** 표시 (인지용)
- 발주 완료 모달 (주문번호 + 출고일/도착일 안내 + 계속발주/발주내역 보기)

### 6. 발주내역
- 목록 보기 / 매장별 보기 전환 (신화푸드는 매장별 기본)
- 상태 필터 (전체/대기/확인/완료/취소)
- 주문 클릭 → 상세 모달 (상품 목록, 합계)
- **주문 상태**: 대기(pending) → 확정(confirmed) 2단계 (shipping/delivered 미사용)
- **마감 전 + 대기 상태**: 가맹점이 본인 주문 수정/취소 가능 (예치금 자동 환불)
- **마감 후**: 가맹점 수정/취소 차단, 관리자만 가능
- **신화푸드**: 주문 수정/취소 권한 없음, **확정 가능** (pending → confirmed)
- **관리자**: 모든 상태에서 수정/취소/확정 가능
- **확정 처리**: 관리자 + 신화푸드 (pending → confirmed)
- **결품 시**: 관리자가 주문 수량 수정 또는 재고 수동 조정
- **변경 이력**: 누가(이름+역할), 언제, 왜 수정했는지 기록/표시
- **거래명세서 출력**: 주문 상세에서 버튼 클릭 → 새 창 → 인쇄/PDF 저장

### 7. 거래명세서
- 주문별 거래명세서 출력 (`/statement/[주문ID]`)
- 배송/실무용 심플 양식: 주문정보 + 배송처(매장명/주소/담당자/연락처) + 품목 + 합계
- 하단에 산방에프앤비, 신화푸드 연락처 표시
- 관리자, 신화푸드, 가맹점 모두 출력 가능
- 브라우저 인쇄 → PDF 저장 지원

### 8. 예치금관리
- 입금 계좌 안내 상시 표시 (하나은행 776-910015-28704, 산방에프앤비)
- 관리자: 가맹점 선택 → 입금/출금/조정 (콤마 구분 + 한글 금액 표시)
- 가맹점: 자기 매장 잔액 조회 + 거래 내역 (조정 버튼 없음)
- **입금 요청 플로우**: 가맹점이 계좌 입금 후 "입금했어요" 버튼 → 관리자 승인/반려
- **입금일자 메모 필수** (예: "3/24입금") → 관리자가 계좌 확인 시 참고
- 발주 시 자동 차감, 취소 시 자동 환불

### 9. 재고관리
- 전용상품: 관리자만 관리 (비밀번호 재확인 — 구현 예정)
- 범용상품: 관리자 + 신화푸드 관리 가능
- 재고 카드 (수량 + 부족 알림)
- 입고/출고/조정 등록
- 상품별 입출고 이력 필터
- 발주 시 자동 차감, 취소 시 자동 복구, 수정 시 차이 반영

### 10. 정산관리 (관리자 전용)
- 월 선택 (연/월)
- **정산 요약**: 받은 돈(예치금) / 직영점 후불 / 줘야 할 돈 / 순이익
- **계산서 체크리스트**: 발행/수신 항목 자동 계산
- **가맹점별 상세 테이블** (전용/범용/합계)
- **엑셀(CSV) 다운로드**

### 11. 공지사항
- 관리자: 작성/수정/삭제, 상단 고정
- 대상 선택: 전체 매장 또는 특정 매장 지정
- 가맹점: 전체 공지 + 자기 매장 대상 공지만 표시
- 대시보드에 최신 5건 미리보기

### 12. UI/UX
- 이모지/색상 단순화 (발주프로그램에 맞게 실용적으로)
- 글씨 크기 확대 (모바일 퍼스트)
- 반응형 웹 (PC + 모바일)
